# Webapp — AI Agent Instructions

React 19 frontend for LoMo. See the root [AGENTS.md](../../AGENTS.md) for repo-wide info.

## Code Style

- **Indentation**: tabs (not spaces)
- **Quotes**: double quotes
- **Semicolons**: always
- **ESLint preset**: `@antfu/eslint-config` with `formatters: true` (shared from `@repo/eslint-config`)
- **No Prettier** — formatting is handled by ESLint via `eslint-plugin-format`
- **Import sorting**: handled by ESLint (antfu sorts imports); `source.organizeImports` is disabled in VSCode
- Always run `bun run lint:fix` from the repo root after making changes

## Stack

- React 19 with React Compiler (`babel-plugin-react-compiler`)
- TypeScript strict mode: `noUnusedLocals`, `noUnusedParameters`, `verbatimModuleSyntax`
- Vite 7 (build tool)
- Styling: Tailwind CSS (coming soon — not yet installed)
- Component library: TBD (shadcn/ui or custom React Aria components)
- State management: TanStack Query planned, not yet installed

## Routing

TanStack Router with file-based routing in `src/routes/`.

- Route convention: `export const Route = createFileRoute("/path")({ component: Name })`
- Root layout: `src/routes/__root.tsx` using `createRootRoute`
- Route tree is auto-generated into `routeTree.gen.ts` by the Vite plugin

## Testing

No test runner configured yet. Vitest will be used when added.

## Do NOT

- Do NOT edit `src/routeTree.gen.ts` — auto-generated by TanStack Router plugin
- Do NOT add Prettier or any standalone formatting tool — ESLint handles formatting
- Do NOT use spaces for indentation — the project uses tabs
